## Welcome

Welcome to the GitHub organization of the **Laboratório de Imunoinflamação**, based at the **Universidade Estadual de Campinas (UNICAMP)**, within the **Instituto de Biologia** and the **Departamento de Microbiologia e Imunologia**.

This space brings together code, analysis workflows, notebooks, reports, and documentation developed by the laboratory to support reproducible and collaborative research in immunology, inflammation, host–microbe interactions, and biomedical data analysis.

## Purpose

This organization is intended to provide a structured environment for:

- computational research projects;
- bioinformatics and data analysis workflows;
- internal documentation;
- collaborative development;
- long-term organization of scientific code and reports.

## Guidelines

For repository organization, Git/GitHub usage, data management, and contribution practices, please refer to the laboratory documentation/wiki.

## Contact

For questions about repository access, organization, or computational workflows, please contact the repository maintainer or the organization administrators.
